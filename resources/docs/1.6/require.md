<a name="section-1"></a>
## 必备资源
* VPS一台(最低1核1G)
* 域名一个(可选)
* LNMP环境(Linux、Nginx、Mysql、PHP)  

> {warning} 没有云服务器可以前往阿里云、腾讯云、百度云、华为云等云服务器厂商购买，不推荐使用虚拟主机。  
国内开设网站需要ICP备案、公安网备案，不想备案可以使用香港VPS或者国外VPS。

<a name="section-2"></a>
## 环境要求
* Nginx >= 1.8
* PHP >= 7.1.3
* Mysql >= 5.6
* Redis >= 3.0
* Sqlite PHP 拓展
* GD PHP 拓展
* OpenSSL PHP 拓展
* PDO PHP 拓展
* Mbstring PHP 拓展
* Tokenizer PHP 拓展
* XML PHP 拓展
* Ctype PHP 拓展
* JSON PHP 拓展
* BCMath PHP 拓展
* Redis PHP 扩展

> {warning} 如果您已安装了LNMP环境可跳过下一节直接进入[安装酷博](/{{route}}/{{version}}/setup)章节
